# openconstruct-abi

A [SuperInstance](https://github.com/SuperInstance) crate.

## Ecosystem Integration

- Stable ABI layer that wraps `OpenConstruct` for consumption from any language via FFI
- Used by all polyglot bindings — Python, JavaScript, C, and future language SDKs
- Integrates with `plato-room` through the OpenConstruct room definition pipeline
- Provides `a2ui-render` compatible output for visualization of constructed geometry
- Central to the buildability stack — makes constructive geometry accessible ecosystem-wide
